```bash
parsnp -r /home/username/group4/input-sequences-final/D0287766_S01_L001-SKESA_contigs.fasta \
       -d /home/username/group4/input-sequences-final \

```
- **Output:**
- ### 📄 `parsnp.xmfa`
- **Description**: XMFA (Multiple Alignment Format) file representing the core genome alignment of all input genomes.
- **Usage**: Can be viewed with alignment viewers or converted to other formats. Required for downstream visualization and analysis (e.g., iTOL or Gingr).
- ### 🌳 `parsnp.tree`
- **Description**: Newick-format phylogenetic tree based on the core genome alignment.
- **Usage**: Input for tree visualization tools (e.g., FigTree, iTOL). Shows the evolutionary relationships among input genomes.
- ### 📦 `parsnp.ggr`
- **Description**: Binary compressed core genome representation.
- **Usage**: For internal use by Gingr for real-time, interactive exploration of genome alignments and variants.
- ### `parsnpAligner.log`
- **Description**: Log file capturing runtime details, process status, and any warnings or errors during execution.
- **Usage**: Helps with debugging, reviewing runtime parameters, or verifying reproducibility.
- ### ⚙️ `parsnpAligner.ini`
- **Description**: Configuration file generated by Parsnp to document the parameters and settings used in the run.
- **Usage**: Essential for reproducibility and understanding specific aligner behavior.
# Gubbins

This tool identifies regions of recombination in bacterial whole genome alignments.

## System Requirements

### Architecture
- x86_64 (Intel)

### Operating System
- macOS

### Dependencies
Make sure the following software and libraries are installed:
- Python 3.9
- Gubbins
- HarvestTools (part of the Harvestsuite)
- Parsnp (to generate alignments)
- RAxML or FastTree (used internally by Gubbins)

You can install Gubbins via:

```bash
conda install -c bioconda -c conda-forge --no-channel-priority gubbins python=3.9
```

## Inputs

Before running Gubbins, you must first generate a whole genome alignment using **Parsnp**:

- `parsnp.xmfa`: Multiple alignment file in XMFA format.
- `parsnp.aln`: Concatenated alignment in multi-FASTA format, used as input for Gubbins.

## Outputs

Gubbins produces the following outputs in the working directory:

- `parsnp.filtered_polymorphic_sites.fasta`: Alignment after filtering recombination.
- `parsnp.final_tree.tre`: Phylogenetic tree built after recombination filtering.
- `parsnp.recombination_predictions.gff`: GFF file with identified recombination events.
- `parsnp.node_labelled.final_tree.tre`: Tree with internal node labels.
- Summary statistics in various tabular formats (vcf, tsv, csv).

## Commands Used

### Step 1: Convert XMFA to aligned FASTA
```bash
harvesttools -x parsnp_outdir/parsnp.xmfa -M parsnp_outdir/parsnp.aln
```

### Step 2: Run Gubbins on the alignment
```bash
run_gubbins.py parsnp_outdir/parsnp.aln
```

This will launch the full Gubbins pipeline on the aligned sequences, including:
- Phylogenetic tree building 
- Recombination detection
- Filtered alignment and tree output
